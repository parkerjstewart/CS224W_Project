# In this folder
## load_mens_data.py
Script to load data from Jeff Sackmann's tennis_atp dataset:
https://github.com/JeffSackmann/tennis_atp
Gets the top 150 players by "current ranking" updated 2024
Gets the most recent ~6000 singles ATP matches between players in this top 150

## matches_raw.csv
~6000 matches info. Lots of unnecessary columns here

## top150_players_raw.csv
Top 150 players by ranking as of end of season 2024. Most columns are useful but not all
